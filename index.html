Setup Guide
Connect to Google Sheets in 5 minutes
1

Create a Google Sheet

Go to sheets.google.com and create a new spreadsheet. Name it MA Contract Cost Tracker. Leave it open.

2

Open Apps Script

In your Google Sheet, click Extensions → Apps Script. A new tab will open with a code editor.

3

Paste the backend code

Delete everything in the editor. Open the file Code.gs (provided alongside this app), copy its entire contents, and paste it into the Apps Script editor. Click 💾 Save.

4

Deploy as Web App

Click Deploy → New deployment. Set type to Web app. Set Execute as: Me. Set Who has access: Anyone. Click Deploy and authorize when prompted.

5

Copy the Web App URL

After deploying, copy the Web App URL. It looks like:

https://script.google.com/macros/s/AKfy.../exec
6

Connect this app

Paste the URL into the Setup required bar at the top of this page and click Connect. Your data will sync with Google Sheets in real time — accessible from any device!

💡 Tip: Share the Google Sheet with your team. Everyone who uses this app with the same URL will read and write to the same sheet.
⚠ Note: Data is stored in your Google account. This app never sends data anywhere else.
